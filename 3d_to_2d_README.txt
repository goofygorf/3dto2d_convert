3d point to 2d pixel README

2 text files: 
 3d_point_to_2d_pixel : primary utility MEL script

 dsPoint2PixelDoc.mel : user documentation text file

save both files into your Maya scripts directory or which ever directory file path you have setin Maya to read external user scripts.  To use, enter this command into the MEL command prompt: source "3d_point_to_2d_pixel.mel";   Or create a custom Shelf Button with the same command.

last tested in Maya 2018

User demonstration video: https://youtu.be/cDlkomD_9xo
